Como usar este wireframe
  1. Abrir no browser: /site/esbocos/wireframe-home-v1.html
  2. Replicar no Stitch/Figma frame a frame (hero primeiro).
  3. Exportar mockup limpo do wizard (sem sidebar Console).
  4. Só então atualizar o React da home para casar 1:1 com este layout.
  5. Proibido no hero: print com sidebar admin, “Dados demo”, filmstrip de 3 telas.

Stitch prompt: copiar o bloco “Prompt pronto” da conversa de design, anexar este HTML como referência de estrutura.
